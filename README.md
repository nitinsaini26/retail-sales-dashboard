# India Retail Sales Analytics Dashboard (FY 2025)

An interactive and responsive sales analytics dashboard built using HTML, CSS, JavaScript, and Chart.js, with AI-assisted development using Claude.

## Features
- Dynamic KPIs (Sales, Profit, Orders, AOV)
- Interactive Filters (City, Category, Sales Rep, Date)
- Monthly Sales Trends
- City-wise & Category Performance
- Profitability & Discount Analysis
- Top & Bottom Products
- Payment Method Insights

##  Tech Stack
- HTML, CSS, JavaScript
- Chart.js
- Claude AI (for faster development & UI optimization)

## 📌 Project Purpose
This project demonstrates how raw retail data can be transformed into actionable business insights using data visualization and dashboard design principles.
